#4. Run a Color Run 5K 2018

Located in just down the road in Bangor Maine is the color run. The color run also known as "the happiest 5,000 meters on the planet" is an untimed fun run with runners ending up covered in colored powder. All proceeds stay in Maine to help support those affected by pancreatic cancer. Each color thrown represents different forms of cancer.
